# Gán giá trị của một biến cho một biến khác

---

## 1. Ví dụ:

```js
var myVar;
myVar = 5;
// Biến myVar có giá trị là 5

var myNum;
myNum = myVar;
// Biến myNum có giá trị bằng biến myVar là 5
```

## 2. Bài tập:

- Cho mẫu sau:

```js
var a;
a = 7;
var b;
```

- Gán giá trị của biến `a` cho biến `b`
