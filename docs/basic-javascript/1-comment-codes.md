# Comment code trong Javascript

---

## 1. Comment một dòng:

```js
// Đây là comment một dòng
```

## 2. Comment nhiều dòng:

```js
/* Đây là comment
nhiều dòng */
```

hoặc

```js
/**
 * Một
 * Hai
 * Ba
 */
```

## 3. Bài tập:

- Tạo một comment giống với mẫu sau:

```js
/**
 * Your test output will go here
 */
```
